Python

Robotics Operating System

I have been a ROS user for 1 year. ROS is a message passing framework that allows communication between the different subunits of a robot. I used ROS for my research in surgical robotics and for a project on autonomous driving vehicle (BARC). I am familiar with catkin package structure and several open sourced packages in motion planning such as Open Motion Planning Library (OMPL), Openrave & Gazebo

~

Key Packages

Gazebo ~ Openrave ~ OMPL ~ URDF ~ XACRO ~ Rqt

Solidworks

I have experience in CAD deign for 1 year. I took a class in CAD design during my exhange programme in Berkeley. I used Solidworks for design, 3D printing and structural analysis. Some of my projects involve designing an aerodynamically optimized shell for the Berkeley Human Powered Vehicle (HPV) and a miniature Windmill.


Matlab

I have been a Matlab user for 3 years. I used Matlab mostly for data processing & system modelling. Key projects that I did include developing a Finite Element model for wave propagation, signal processing for spatial aliasing of underwater waves, big data analysis of molecular dynamic simulation results and Gaussian process classification for traffic density analysis.

~

Key Libraries

Signal processing ~ Image processing ~ Gaussian process ~ Optimization ~ Finite element modelling

Linux

Arduino

I have been prototyping for 3 years using Arduino with C programming environment. My goal is to enter the Maker community space, using sensors with the cloud to develop solutions for everyday problems. Some of my projects involve a low level servo control for 2 axis gimbal, an automatic door opener using finger print sensing and the development of an IoT inspired smart home.
